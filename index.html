      👤 Administradores

        💡 Alterações são salvas via API. Se a API não estiver configurada, edite diretamente a planilha Config no Google Sheets.

        🔐 Login

        Insira seu alias Amazon (sem espaços, minúsculo), selecione sua DS e digite o PIN de 4 dígitos configurado pelo admin.

        📱 Escaneamento (Mobile)

        A câmera traseira abre automaticamente. Posicione o QR/código de barras dentro da mira. O registro é automático para Bags e Ráfias no modo RETORNO.

        🖥️ Escaneamento (Desktop/USB)

        Em telas maiores que 768px sem touch, o modo USB é ativado. O campo de texto recebe foco automático. Basta ler o código com o scanner USB.

        📦 Tipos de Ativos

        • Bag: Sacolas de entrega. Scan rápido contínuo.
        • Ráfia: Sacos grandes de transporte. Scan rápido contínuo.
        • Pacote (TBR): Pacotes com problema. Requer classificação e permite foto.

        📊 Dashboard

        Visualize métricas de saídas, retornos e pendências. Filtre por DS, tipo e data.

        ⚙️ Admin

        Configure a URL da API, gerencie DS, PIN, classificações e administradores.

        🔧 Troubleshooting ▼

        • Câmera não abre: Verifique permissões do navegador.
        • API não conecta: Verifique a URL no Admin e se o script está publicado como webapp.
        • Scan não registra: Código pode estar no cooldown de 5s (anti-duplicata).
        • Dados offline: Registros ficam em fila local e são enviados quando a conexão voltar.

        🔐 Login

        Enter your Amazon alias (no spaces, lowercase), select your DS and enter the 4-digit PIN configured by admin.

        📱 Scanning (Mobile)

        The rear camera opens automatically. Position the QR/barcode within the crosshairs. Registration is automatic for Bags and Ráfias in RETORNO mode.

        🖥️ Scanning (Desktop/USB)

        On screens larger than 768px without touch, USB mode is activated. The text field auto-focuses. Just scan with your USB scanner.

        📦 Asset Types

        • Bag: Delivery bags. Continuous rapid scan.
        • Ráfia: Large transport sacks. Continuous rapid scan.
        • Package (TBR): Problem packages. Requires classification and allows photo.

        📊 Dashboard

        View metrics for outputs, returns and pending items. Filter by DS, type and date.

        ⚙️ Admin

        Configure API URL, manage DS, PIN, classifications and administrators.

        🔧 Troubleshooting ▼

        • Camera won't open: Check browser permissions.
        • API won't connect: Check the URL in Admin and that the script is published as webapp.
        • Scan won't register: Code might be in 5s cooldown (anti-duplicate).
        • Offline data: Records are queued locally and sent when connection returns.
